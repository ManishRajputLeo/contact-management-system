# contact-management-system
A Contact Management System built with Spring Boot, Maven, JPA, and Spring Security is a web application for storing and managing contacts efficiently. It allows users to add, update, delete, and search contact details, while ensuring secure access through authentication and authorization.
Contact Management System

A simple web application to manage contacts, built using Spring Boot, Maven, JPA, and Spring Security.

**Features**
Add, update, delete contacts
Search and view contact details
User authentication and security

**Tech Stack**
Spring Boot
Maven
JPA (Hibernate)
Spring Security
